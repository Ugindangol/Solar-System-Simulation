# Solar-System-Simulation
This project is a 2D graphical representation that demonstrates the motion of planets around the Sun using fundamental computer graphics concepts and trigonometric equations. The main objective of this project is to visualize planetary motion in a simple and interactive way while applying core concepts like transformation, animation and rendering.

#Features
Sun at the centre with glowing effect
Eight planets revolving in elliptical orbits
Moon revolving around Earth (hierarchical motion)
Twinkling background stars
Shooting star animation
User-controlled speed (+ and - keys)
Smooth animation using double buffering

#Graphics Algorithms
Bresenham’s Line Algorithm → Shooting star
Midpoint Circle Algorithm → Sun & planets
Midpoint Ellipse Algorithm → Orbits & Saturn rings

#Requirements
Turbo C / DOSBox / WinBGI
C Compiler supporting graphics.h

#How to Run
Open Turbo C / your graphics-supported C environment
Copy the code into a .c file
Compile and run

#Limitations
Only revolution is implemented (not actual axial rotation)
Not scaled to real astronomical distances
Uses Turbo C graphics (not modern graphics libraries)
